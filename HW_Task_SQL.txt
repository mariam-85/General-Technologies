Task1.

Вывести все (только) имена клиентов.

SELECT ContactName
FROM Customers;


Task 2.

Вывести клиентов из Берлина, имена которых содержат слово "del".

SELECT *
FROM Customers
WHERE ContactName LIKE ('%del')
AND City = 'Berlin';


Task 3.

Вывести контактные имена клиентов из Лондона.

SELECT ContactName
FROM Customers
WHERE City = 'London';
